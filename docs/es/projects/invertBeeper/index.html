Invertir cono (con solución)

Credit: Basado en un entregable por Eric Roberts

Handouts: Karel Reference
File: invert_beeper.py

En un mundo de cualquier tamaño, con cualquier configuración de conos (donde ningun cuadro tendrá más de uno), invierte todos los espacios con conos, de forma que donde había un cono, ya no haya uno y viceversa, que donde no había un cono, haya uno. Como se muestra en el siguiente ejemplo:

Solution

Show Solution 

from karel.stanfordkarel import *
from templates.es.projects.stepUp.soln import girar_derecha

"""
Invierte todos los conos, para que donde había anteriormente un cono
ya no haya ninguno... Y donde no había cono, ahora haya uno.
"""


def main():
    # Siempre hay por lo menos una linea en el mundo.
    invertir_fila()
    volver_al_oeste()

    # Invierte las lineas despejadas
    while izquierda_despejada():
        ir_a_siguiente_fila()
        invertir_fila()
        volver_al_oeste()


def ir_a_siguiente_fila():
    """
    Karel se mueve una fila. Asume que hay una fila encima de Karel,
    y que Karel esta mirando al este. Cuando acaba esta funcion, 
    Karel estará una fila encima, mirando al este.
    """
    girar_izquierda()
    moverse()
    girar_derecha()


def invertir_fila():
    """
    Invierte una sola fila, al inicio de la ejecución, Karel debería estar
    viendo al este, desde la parte oeste de la fila. Después, Karel debería 
    estar mirando al este del lado derecho de la misma linea, y todos los 
    conos estarían invertidos con invertir_cono()
    """
    while frente_despejado():
        invertir_cono()
        moverse()
    invertir_cono()


def invertir_cono():
    """
    Invierte la configuración de conos. Si había un cono, lo recogerá, si
    no hay ninguno, se pondrá un cono.
    """
    if conos_presentes():
        recoger_cono()
    else:
        poner_cono()


def volver_al_oeste():
    """
    Karel se da la vuelta y vuelve al muro en esa dirección. Tras
    esta función, Karel estará mirando la  misma dirección a la que 
    estaba mirando originalmente.
    """
    dar_la_vuelta()
    while frente_despejado():
        moverse()
    dar_la_vuelta()


def dar_la_vuelta():
    """
    Karel da una vuelta de 180 grados, si estaba viendo al este, ahora verá
    al oeste, si estaba viendo al norte, ahora verá al sur (y viceversa). 
    """
    girar_izquierda()
    girar_izquierda()


def turn_right():
    """
    Karel gira a la derecha, por ejemplo, si esta mirando al este, Karel
    ahora verá al sur. Si Karel esta viendo al norte, ahora verá al este.
    """
    for i in range(3):
        girar_izquierda()


# No te preocupes por este codigo, no debe modificarse.
if __name__ == "__main__":
    run_karel_program()
